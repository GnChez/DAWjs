1. Afegir Classes

Cada vegada que es pulsa un dels botons s'aplicarà l'estil relacionat al botó a l'element Caixa Afegir.

A mesura que es van pulsant els botons s'aniran mantenint les classes que ja s'estan aplicant.

Al pulsar el botó 'Eliminar tot' es deixaran d'aplicar tots els estils.

La relació dels botons amb les classes de CSS son:
'Afegir Text Verd' -> text-verd
'Afegir Rotació' -> rotacio
'Afegir Marge' -> marge
'Afegir Radius' -> marge-radius

Per fer aquesta activitat s'ha d'utilitzar els mètodes add i remove.

Caixa Afegir

2. Toggle de Classes

Cada vegada que es pula el botó 'Aplica/treu CSS' anirà commutant entre aplica o treure la classes 'activa' al div 'Caixa toggle'

Per defecte no té aplicada la classe 'activa'

Proposta de millora: Canviar el text del botó cada vegada que es pulsa. Si s'aplica l'estil 'activa' el text del botó canvia a 'Treu CSS', en cas contrari el text serà 'Aplica CSS'.

Per fer aquesta activitat s'ha d'utilitzar el mètode toggle. En la proposta de millora necessitareu utilitzar el mètode contains per saber si s'aplica el css i la propietat textContent per canviar el text del botó.

Caixa Toggle

3. Canvi de Classes CSS

Cada vegada que es pulsa el botó 'Canvia classe CSS' se li haurà d'aplicar al div 'Caixa Ciclica' una nova classe CSS

Per defecte té la classe estat-1, al clicar el botó se li aplicarà la classe estat-2 i l'estat-1 deixarà d'aplicar-se, igualment passarà amb l'estat-3 i finalment amb l'estat-4

Quan el div 'Caixa Ciclica' té la classe estat-4 i es pulsa el botó tornarà a aplicat la classe estat-1 i l'estat-4 deixarà d'aplicar-se.

Les classes d'estils per aquesta activitat son: .estat-1, .estat-2, .estat-3 i .estat-4

S'ha d'anar mostrant l'estat actual a través d'un number

Per fer aquesta activitat s'ha d'utilitzar els mètodes add i remove.

Caixa Ciclica
Estat actual: 1
